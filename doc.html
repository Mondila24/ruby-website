Introduction

The Loruki platform is a cloud hosting platform that allows you to deploy and manage your applications with ease. It provides a range of features and tools to help you build, scale, and monitor your applications in the cloud.

Install CLI

Requirements

  • Windows 10, Mac OSX, Linux
  • Node.js 12 or higher

Install

Mac (Homebrew)

$ brew install Loruki-Cli

NPM

$ npm install Loruki-Cli

Yarn

$ yarn install Loruki-Cli
